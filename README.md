This program asks the user to enter their name, school, interests and internship goal. After the user inputs their information, it's displayed back to them.

This program also demonstarted the basics of python programming by:

- collecting user input with 'input()'
-  using variables
-  Displaying output with 'print()'

  How to get started:
  - Open the file 'student intro2.py' in python IDLE
  - Run the program and enter the requested info
  - Program will display the information entered
